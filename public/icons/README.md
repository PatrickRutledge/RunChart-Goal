# PWA Icons Directory

This directory should contain the following PWA icons for Microsoft Store submission:

## Required Icons

### Standard PWA Icons
- `pwa-192x192.png` - 192x192 px
- `pwa-512x512.png` - 512x512 px

### Microsoft Store Icons
- `StoreLogo.png` - 50x50 px
- `Square150x150Logo.png` - 150x150 px
- `Square44x44Logo.png` - 44x44 px
- `Wide310x150Logo.png` - 310x150 px

### Optional Assets
- `apple-touch-icon.png` - 180x180 px (iOS)
- `masked-icon.svg` - SVG with mask support
- `screenshot-wide.png` - 1280x720 px (Desktop screenshot)
- `screenshot-narrow.png` - 750x1334 px (Mobile screenshot)

## Icon Guidelines

- Use high-quality, crisp graphics
- Ensure icons work on both light and dark backgrounds
- Follow Microsoft Store design guidelines
- Include your app branding/logo
- Test icons at various sizes

## Generating Icons

You can use tools like:
- [PWA Builder](https://www.pwabuilder.com/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Adobe Illustrator/Photoshop
- Figma with export options

Replace this README with actual icon files before publishing.
